Module geodsolve

Wrapper to invoke Karney's GeodSolve utility as an (exact) geodesic, but intended for testing purposes only.

Set env variable PYGEODESY_GEODSOLVE to the (fully qualified) path of the GeodSolve executable.

Version: 22.10.11
