                   NearestOn6Tuple

6-Tuple (closest, distance, fi, j, start, end) with the closest point, the distance in meter, conventionally and the start and end point of the path or polygon edge. Fractional index fi (an FIx instance) and index j indicate the path or polygon edge and the fraction along that edge with the closest point. The start and end points may differ from the given path or polygon points at indices fi respectively j, when unrolled (wrap is True). Also, the start and/or end point may be the same instance as the closest point, for example when the very first path or polygon point is the nearest.
