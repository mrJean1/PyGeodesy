toRefFrame (self, point, reframe2, **epoch_epoch2_name)

 

Convert an ellipsoidal point from this reframe and from epoch to an other reframe2 and epoch2 or epoch.

Arguments:
  • point - The point to convert (LatLon or Cartesian).
  • reframe2 - Reference frame to convert to (RefFrame).
  • epoch_epoch2_name - Optional keyword arguments epoch=None, epoch2=None and name=reframe2.name.
Returns:
A copy of the point, converted and renamed.
Raises:
  • TypeError - Invalid point.

Note: The point.reframe is overridden by this reframe and point.epoch by epoch or this reframe.epoch.

See Also: Methods LatLon.toRefFrame and Cartesian.toRefFrame for more details.

Xforms (self, inverse=False)

 

Return all Xforms converting from or to this reference frame or both.

Arguments:
  • inverse - If True, get all from otherwise to Xforms un-inverted (bool) or if inverse=2 (int) get all to Xforms inverted plus all froms.
Returns:
