Class BooleanGH

                  BooleanGH

Polygon class providing boolean operations between two polygons based on clipGH4. The supported operations between polygon A and B are:

  • C = A | B or A |= B, union of A and B
  • C = A & B or A &= B, intersection of A and B
  • C = A - B or A -= B, difference A less B
  • C = B - A or B -= A, difference B less A
