__init__ (self, a_triax, b=None, c=None, name='')
(Constructor)

 

New ordered Triaxial.

Arguments:
  • a_triax - Largest semi-axis (scalar, conventionally in meter) or an other Triaxial or Triaxial_ instance.
  • b - Middle semi-axis (meter, same units as a), required if a_triax is scalar, ignored otherwise.
  • c - Smallest semi-axis (meter, same units as a), required if a_triax is scalar, ignored otherwise.
  • name - Optional name (str).
Raises:
  • TriaxialError - Semi-axes not ordered, spherical or invalid.
Overrides: object.__init__

Note: The semi-axes must be ordered as a >= b >= c > 0 and must be ellipsoidal, a > c.

Convert ellipsoidal lat- and longitude beta, omega and height to cartesian.

Convert ellipsoidal lat- and longitude beta and omega to cartesian coordinates on the triaxial's surface.

Arguments:
  • sbeta - Ellipsoidal latitude sin(beta) (scalar).
  • cbeta - Ellipsoidal latitude cos(beta) (scalar).
  • somega - Ellipsoidal longitude sin(omega) (scalar).
  • comega - Ellipsoidal longitude cos(omega) (scalar).
  • name - Optional name (str).
Returns:
A Vector3Tuple(x, y, z) on the surface.
Raises:
  • TriaxialError - This triaxial is near-spherical.

reverseCartesian (self, x, y, z, h, normal=True, eps=4.440892098500626e-12, name='')

 

"Unproject" a cartesian on to a cartesion off this triaxial's surface.

Arguments:
  • x - X coordinate along a-axis (meter, same units as the axes).
  • y - Y coordinate along b-axis (meter, same units as x).
  • z - Z coordinate along c-axis (meter, same units as x).
  • h - Height above or below this triaxial's surface (meter, same units as x).
  • normal - If True the height is normal to the surface, otherwise radially to the center of this triaxial (bool).
  • eps - Tolerance for surface test (scalar).
  • name - Optional name (str).
Returns:
A Vector3Tuple(x, y, z).

See Also: Methods Triaxial.forwardCartesian and Triaxial.height4.
