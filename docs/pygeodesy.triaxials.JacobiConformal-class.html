                          JacobiConformal

This is a conformal projection of a triaxial ellipsoid to a plane in which the X and Y grid lines are straight.

Ellipsoidal coordinates beta and omega are converted to Jacobi Conformal y respectively x separately. Jacobi's coordinates have been multiplied by sqrt(a**2 - c**2) / (2 * b) so that the customary results are returned in the case of an ellipsoid of revolution (or a sphere, currently not supported).

Note: This constructor can not be used to specify a sphere.
