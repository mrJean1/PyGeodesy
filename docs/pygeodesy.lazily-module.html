Module lazily

Lazily import pygeodesy modules and attributes.

Lazy import is supported only for Python 3.7+ and is enabled by default in PyGeodesy 18.11.10+.

To disable lazy import set environment variable PYGEODESY_LAZY_IMPORT to 0 or an empty string. Use 2 or higher to print a message for each lazily imported module and attribute, similar to environment variable PYTHONVERBOSE showing all imports.

Lazy import only applies to top-level modules of pygeodesy. A lazy import of a top-level module also loads all sub-modules imported by that top-level module.

Based on modutil.lazy_import by Brett Cannon's modutil.
