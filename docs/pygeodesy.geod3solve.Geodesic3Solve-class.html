Wrapper to invoke Karney's Geod3Solve as a triaxial version of Karney's Python class Geodesic.

Notes:
  • Use property GeodSolve or env variable PYGEODESY_GEOD3SOLVE to specify the (fully qualified) path to the Geod3Solve executable.
  • This geodesic is intended for testing purposes only, it invokes the GeodSolve executable for every method call.
