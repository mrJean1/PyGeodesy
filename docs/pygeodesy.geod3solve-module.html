Module geod3solve

Wrapper to invoke Karney's Geod3Solve utility as a triaxial geodesic, but intended mainly for testing purposes.

Set env variable PYGEODESY_GEOD3SOLVE to the (fully qualified) path of the Geod3Solve executable.

Version: 25.12.12
