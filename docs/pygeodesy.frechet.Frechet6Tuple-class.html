                   Frechet6Tuple

6-Tuple (fd, fi1, fi2, r, n, units) with the discrete Fréchet distance fd, fractional indices fi1 and fi2 as FIx, the recursion depth r, the number of distances computed n and the units class or class or name of the distance units.

If fractional indices fi1 and fi2 are int, the returned fd is the distance between points1[fi1] and points2[fi2]. For float indices, the distance is between an intermediate point along points1[int(fi1)] and points1[int(fi1) + 1] respectively an intermediate point along points2[int(fi2)] and points2[int(fi2) + 1].

Use function fractional to compute the point at a fractional index.
