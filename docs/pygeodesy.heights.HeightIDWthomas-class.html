Height interpolator using Inverse Distance Weighting (IDW) and the angular distance in radians from function pygeodesy.thomas_.

See Also: HeightIDWcosineAndoyerLambert, HeightIDWcosineForsytheAndoyerLambert, HeightIDWdistanceTo, HeightIDWflatLocal, HeightIDWhubeny, IDW and SHEPARD_INTERP_2D.
